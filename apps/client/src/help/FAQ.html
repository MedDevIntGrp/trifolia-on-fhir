FAQ

General

My implementation guide has pages, but the table of contents is empty?

Make sure that the "Table of Contents" page's "Auto Generate Table of Contents?" field is set to "Yes".

Home Screen stuck on 'Loading'

Based on your web browser storage settings, Trifolia on FHIR may 'hang" or "stall' at the Loading page with the orange circle. Clearing historical browsing information typically resolves this issue. Please follow the below steps based on the web browser you are using to access Trifolia on FHIR; if the issue is not resolved by the below steps, please submit a ticket and specify which browser you are using.


If using Chrome:

- Select Options (three vertical dots to right of URL address bar within Chrome) and select 'History'

- Open History page and select 'Clear Browsing data' from left pane

- Refresh the page


If using Internet Explorer:

- Select the gear icon (to the right of the URL address bar within IE) and select 'Internet Options'

- On the General Tab, click 'Delete...' button under Browsing History

- Apply > OK


If using Safari:

- Select Develop (drop down top top) then 'Show JavaScript Console' (or Option-Command-C)

- Select Storage Tab

- Highlight Local Storage - trifolia-fhir.lantanagroup.com and Application Cache

- Select each of the key values and press Delete key to remove each of them

- After having removed each key/value pair, refresh the page


If using Microsoft Edge

- Select three horizontal dots (to the right of the URL address bar within Edge)

- History > Clear Browsing Data (Ctrl + Shift + Delete)
